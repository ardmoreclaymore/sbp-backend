# Source and Accuracy Notes

## Data sources

- Hypixel Bazaar API for Bazaar products.
- Hypixel SkyBlock Auction API for active BIN Auction House listings.
- SkyBlock.Tools election page for public mayor/election context.
- OutcroCalculator for public meta/MMM context where available.

## Prediction certainty

Prediction certainty is not a guarantee. It is a probability-style score built from price movement, spread, volume, listing depth, manipulation risk, factor-bank matches, and historical snapshots once enough data exists.

## Current limitation

Until enough historical snapshots and backtest rows exist, prediction output is heuristic. Treat it as a structured forecast, not guaranteed profit advice.
